
# SHA Web

Sitio web oficial de SHA.

Este proyecto tiene como objetivo construir una plataforma web moderna para presentar SHA como una solución profesional de automatización operativa enfocada en seguridad, disciplina y trazabilidad.

---

## Estado del proyecto

⚡ **En etapa de desarrollo activo.**

Pasamos la fase de planificación y actualmente la arquitectura base ya está desplegada y funcional, priorizando el rendimiento y la modularidad.

- **Versión:** 0.2
- **Manejador de paquetes:** `pnpm` (optimización de dependencias).

---

## Objetivos

- Construir una Landing Page premium.
- Generar confianza desde el primer segundo.
- Explicar el funcionamiento de SHA.
- Mostrar el producto sin revelar la lógica interna.
- Obtener solicitudes de demostración.

## 📁 Estructura Desarrollada

- `app/layout.tsx`: Control global de layouts, fuentes y Metadata/SEO para indexación.
- `app/page.tsx`: Landing Page principal (Home).
- `app/contacto/page.tsx`: Formulario de solicitud de acceso con validación asíncrona avanzada.
-`app/como-funciona/page.tsx`: Explicación del flujo técnico y de ejecución asíncrona de las órdenes matemáticas.
-`app/legal/page.tsx`: Términos de servicio, descargo de responsabilidad sobre trading (Risk Disclaimer) y políticas de privacidad.
-`app/planes/page.tsx`: Presentación de la oferta comercial y niveles de acceso a la plataforma de software (en fase de definición).
-`app/seguridad/page.tsx`: Detalle de la arquitectura no-custodial y buenas prácticas para el manejo seguro de llaves API.
-`app/sobre-sha/page.tsx`: Filosofía de SHA, misión del equipo y enfoque técnico sobre la disciplina de capital y control de riesgos.
- `app/faq/page.tsx`: Sección de transparencia informativa sobre custodia de fondos y uso seguro de APIs.
- `components/ui/`: Inputs y áreas de texto reutilizables abstraídos mediante `React.forwardRef`.
